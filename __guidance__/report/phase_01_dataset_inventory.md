## Reconciliation Checks

Passed checks:

- Raw videos: 44 MP4 files and 44 sidecar JSON files.
- Annotation rows: 36,482 total stroke rows across 104 non-empty set CSVs.
- Feature variants: each variant has 33,481 clip branches.
- Feature triples: no missing `joints` / `pos` / `shuttle` files.
- Class folders: merged variants have 25 classes; original variants have 35 classes.

Known mismatch:

- Annotation stroke rows: 36,482.
- Feature clips per variant: 33,481.
- Difference: 3,001 annotation rows are not represented as feature clips.

This mismatch was already documented in `data.md`; the Phase 01 inventory confirms it still exists in the current checkout.

## Assumptions and Unclear Items

- Unclear from current codebase: the exact filtering rule that maps 36,482 annotation rows to 33,481 feature clips.
- Unclear from current codebase: whether the missing 3,001 annotation rows are due to dropped unusable clips, missing detections, class filtering, sequence-window constraints, or another upstream preprocessing rule.
- Unclear from current codebase: whether the `.info.json` files are required after acquisition; this phase preserves them as provenance metadata.

## Blockers

No blocker for Phase 02 ground-truth table construction.

Phase 03 has a known prerequisite:

- The feature folders are non-collated per-clip arrays. A collation step is required before `bst_infer.py` can use `Dataset_npy_collated`.
